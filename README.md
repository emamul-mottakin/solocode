## How to run
Please follow the below instructions to run different branches of this repository in your machine:
1. Clone this repository -
    ```sh
    git clone https://github.com/emamul-mottakin/solocode
    ```
2. Install npm
    ```sh
    npm i
    ```
3. Run the project
    ```sh
    npm run dev
    ```
